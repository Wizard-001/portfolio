# Modern AI Portfolio

A high-performance, visually stunning portfolio built with **Next.js 15**, **React 19**, and **Framer Motion**. This project showcases advanced UI/UX patterns, generative AI integrations, and interactive data visualizations.

![Portfolio Preview](https://github.com/Wizard-001/portfolio/raw/main/public/globe.svg) *<!-- Replace with actual screenshot later -->*

## 🚀 Features

- **Dynamic Animations**: Powered by `Framer Motion` and `motion` for fluid, high-fidelity transitions.
- **Interactive Data Viz**: Custom D3.js skill visualizations and orbital timelines.
- **Particle Systems**: Advanced background effects using `@tsparticles`.
- **Modern UI Components**: Built with `Shadcn UI`, `Radix UI`, and `Lucide React`.
- **Next.js 15+**: Leveraging the latest App Router features and React 19 capabilities.
- **Responsive Design**: Fully optimized for all device sizes using Tailwind CSS 4.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Visualization**: [D3.js](https://d3js.org/)
- **Icons**: [Lucide](https://lucide.dev/), [Tabler Icons](https://tabler.io/icons)
- **Components**: [Shadcn UI](https://ui.shadcn.com/)

## 📦 Getting Started

### Prerequisites

- Node.js 18+ 
- npm / pnpm / yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Wizard-001/portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
